# Github Issue Tracker Application

A real-time Issue Tracking Dashboard built with a focus on clean UI and seamless user experience. This application allows users to monitor, search, and filter tasks fetched dynamically from an API.



## 🚀 Live Demo
You can view the live project here: [GitHub Live Link](https://lnkd.in/gv7dnGpc)

## ✨ Features

- **Secure Login:** A dedicated and secure sign-in page to protect dashboard access.
- **Real-time Data:** Fetches live issue data directly from an external API for up-to-date information.
- **Smooth UX:** Includes a custom loading spinner to ensure a fluid experience during data fetching and searches.
- **Advanced Filtering:** Easily toggle between **All**, **Open**, and **Closed** status to manage tasks efficiently.
- **Instant Search:** A powerful search bar to find specific issues by title or keyword instantly.
- **Detailed View:** Interactive modals (popups) that display comprehensive details including:
    - Author information
    - Labels
    - Priority levels
- **Fully Responsive:** Optimized with a mobile-first approach, ensuring it looks great on desktops, tablets, and smartphones.

## 🛠️ Technologies Used

- **HTML5:** For semantic structure.
- **Tailwind CSS & DaisyUI:** For modern, responsive styling and pre-built UI components.
- **JavaScript (ES6+):** For API integration, DOM manipulation, and filtering logic.
