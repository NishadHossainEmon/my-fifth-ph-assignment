✅ Features included: 
➤ Easy Login: A secure sign-in page to access the dashboard.
➤ Live Data: It fetches real-time issue data from an API.
➤ Loading Spinner: Added a smooth loading spinner to improve user experience while fetching data or searching.
➤ Filter Issues: You can easily switch between "All", "Open", and "Closed" tasks.
➤ Search Bar: Quickly find any issue by typing its name.
➤ View Details: Click on any issue to see more details like the author, labels, and priority in a popup (modal).
➤ Mobile Friendly: The design works perfectly on both phones and computers.

Technologies I used: HTML ,Tailwind CSS(Daisy UI) ,Javascript
